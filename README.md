Tools Used in the Project:
NextJS 14, MongoDB, EmailJS, TailwindCSS

This is a fully  responsive full-stack project. 

In this project: 

Features:
       User Registration
       User Login (username password or Github)
       Admin page 
          Admin previliges:
             create/delete user
             create blog posts
             delete blog posts
       Error handling
